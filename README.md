# Dla Marty ❤️

Romantyczna strona przygotowana do publikacji na GitHub Pages.

## Jak opublikować

1. Utwórz nowe repozytorium na GitHub, np. `dla-marty`.
2. Wgraj całą zawartość tego folderu.
3. Wejdź w **Settings → Pages**.
4. Wybierz:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Zapisz.

Po chwili strona będzie dostępna pod adresem:
https://TWOJ_LOGIN.github.io/dla-marty/
